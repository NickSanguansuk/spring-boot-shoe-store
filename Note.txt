
[_ Create Spring Boot Project _]

---------- ---------- ---------- ---------- ---------- 

Create New Project -> Spring Initializr

	Name: 		shoe-store
			Java / Maven
	Group:		com.company
	Artifact:	shoe-store
	Package name:	com.company.shoe_store
	SDK:		corretto-11
	Packaging: 	War

	Spring Boot:	2.5.5

Add needed Dependencies & Plugins in POM.xml

Add # Database Setup & # Tomcat configuration (can add everything here) in application.properties

Create index.jsp file in src > main > webapp > WEB-INF > jsp > index.jsp

Create SlashController

Comment out the "spring-boot-starter-security" or (username: user, password: auto-generate)

---> Now run the App, and test in the browser

---------- ---------- ---------- ---------- ---------- 

Link the project to the database "shoe_store dev-mariadb"

Create all entities
Create all dao

---------- ---------- ---------- ---------- ----------

Set up JSP and Thymeleaf together in this Spring Boot project

---------- ---------- ---------- ---------- ----------

[_ Run _]

Run with SpringJspApplication
or
Maven tab (on the right) > Lifecycle > install > Click "Run Maven Build"
After finish, copy the .war file from "target" folder
Put the .war file anywhere we want, then run this command in PowerShell
Run     ---> "java -jar spring-jsp-0.0.1-SNAPSHOT.war"
Stop    ---> Ctrl+C

[_ URL _]

http://localhost:8080
